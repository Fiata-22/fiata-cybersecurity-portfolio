# Cybersecurity Projects

Selamat datang di koleksi proyek cybersecurity saya. Repository ini berisi berbagai proyek yang saya kerjakan untuk meningkatkan kemampuan teknis dalam bidang keamanan siber, analisis jaringan, threat intelligence, dan pengembangan solusi keamanan.

## Tujuan

Tujuan dari proyek-proyek ini adalah:

* Mengembangkan kemampuan analisis keamanan.
* Memahami ancaman siber yang umum terjadi.
* Membuat solusi yang dapat membantu meningkatkan keamanan pengguna.
* Mendokumentasikan proses pembelajaran dan hasil penelitian keamanan.

---

## Project List

### 1. Network Traffic Analysis

**Status:** Completed

Analisis lalu lintas jaringan menggunakan Wireshark untuk mengidentifikasi aktivitas yang mencurigakan, komunikasi protokol, serta potensi ancaman keamanan.

**Skills:**

* Packet Analysis
* Network Monitoring
* Protocol Inspection

**Tools:**

* Wireshark
* Linux

---

### 2. Web Security Assessment

**Status:** Ongoing

Melakukan penilaian keamanan terhadap aplikasi web menggunakan metodologi OWASP Top 10 untuk menemukan dan mendokumentasikan kerentanan yang umum ditemukan.

**Skills:**

* Vulnerability Assessment
* Web Security Testing
* Security Reporting

**Tools:**

* Burp Suite
* OWASP ZAP
* Nmap

---

### 3. Phishing Detection Research

**Status:** Research Phase

Penelitian mengenai karakteristik situs phishing dan metode deteksi dini untuk membantu pengguna mengidentifikasi tautan berbahaya.

**Skills:**

* Threat Intelligence
* Phishing Analysis
* Security Awareness

**Tools:**

* VirusTotal
* URL Analysis
* Open Source Intelligence (OSINT)

---

### 4. Security Automation Scripts

**Status:** Ongoing

Kumpulan script Python dan Bash yang digunakan untuk membantu otomatisasi tugas-tugas keamanan seperti pemeriksaan layanan, pengumpulan informasi sistem, dan analisis log.

* Automation

### 5. Threat Intelligence Collection

### 6. Penetration Testing Lab

**Status:** Ongoing

Simulasi penetration testing pada environment laboratorium untuk memahami proses end-to-end pengujian keamanan sistem, mulai dari reconnaissance hingga reporting.

**Scope:**

- Lab environment (TryHackMe / Hack The Box / DVWA)
- Authorized testing only

**Skills:**

- Reconnaissance
- Enumeration
- Vulnerability Validation
- Exploitation (Controlled Lab)
- Reporting

**Tools:**

- Nmap
- Burp Suite
- Gobuster
- Metasploit (basic)
- Netcat

**Methodology:**

Mengikuti alur standar penetration testing:

1. Information Gathering
2. Service Enumeration
3. Vulnerability Identification
4. Exploitation (lab only)
5. Privilege Analysis
6. Documentation

**Outcome:**

Memahami alur kerja penetration testing secara sistematis dan etis dalam lingkungan yang aman.
